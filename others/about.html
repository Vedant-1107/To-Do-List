About This Project

Welcome to the To-Do List web app! This simple yet powerful task management tool helps you keep track of your daily tasks efficiently.

Features

  • Add and delete tasks easily.
  • Save tasks locally using browser storage.
  • Search through tasks quickly.
  • Responsive and user-friendly design.

Technologies Used

  • HTML, CSS for structure and styling.
  • JavaScript for interactivity and local storage handling.
